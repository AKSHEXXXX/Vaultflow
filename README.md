# SaAS - Spring Boot Demo

A Spring Boot web application built with Maven.

## Prerequisites

- Java 25+
- Maven 3.9+

## Running the Application

```bash
mvn spring-boot:run
```

The app starts at [http://localhost:8080](http://localhost:8080).

## Building

```bash
mvn clean package
```

## Project Structure

```
src/
├── main/
│   ├── java/com/example/demo/
│   │   └── DemoApplication.java
│   └── resources/
│       └── application.properties
└── test/
    └── java/com/example/demo/
        └── DemoApplicationTests.java
```
